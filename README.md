## Customization

You can customize the portfolio by:

1. **Personal Information**: Update contact details in About, Contact, and Footer components
2. **Projects**: Modify the projects array in Projects.js
3. **Skills**: Update the skills array in Skills.js
4. **Colors**: Change the color scheme in CSS files (current: red/dark theme)
5. **Social Links**: Update social media links in Hero and Footer components

## Sections

- **Home/Hero**: Introduction with call-to-action buttons
- **About**: Personal information and education
- **Skills**: Technical and soft skills with visual indicators
- **Experience**: Professional experience timeline
- **Projects**: Featured projects with descriptions
- **Contact**: Contact form and contact information
- **Footer**: Additional links and social media
